## ⚠️ 已知问题

### 页面无法打开故障排查

**故障现象：**
- 优惠配置页面无法访问
- 活动专场页面无法访问

**原因：** Node.js 服务未启动

**解决方法：**
```bash
# 登录阿里云 CMS01 服务器

# 检查 Node.js 进程
ps aux | grep node
# 或使用 pm2
pm2 status

# 启动服务
cd /path/to/project
npm start
# 或
node server.js
# 或
pm2 start app.js
```
