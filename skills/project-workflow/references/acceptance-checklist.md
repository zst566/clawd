# 客户验收检查清单

> 用于系统性地验证系统是否满足客户需求，确保交付质量

---

## 一、准备工作

### 1.1 验收环境
- [ ] 验收环境已准备（生产环境或准生产环境）
- [ ] 测试数据已准备（贴近真实场景）
- [ ] 系统已部署最新版本
- [ ] 访问权限已配置

### 1.2 验收材料
- [ ] 需求文档（作为验收依据）
- [ ] 测试报告
- [ ] 用户操作手册（初版）
- [ ] 验收检查清单（本文档）

---

## 二、功能验收

### 2.1 核心功能
- [ ] 核心业务流程可以完整跑通
- [ ] 每个功能模块的主要功能可用
- [ ] 数据可以正常创建、查询、修改、删除
- [ ] 报表和统计功能数据正确

### 2.2 用户权限
- [ ] 不同角色登录后看到的内容正确
- [ ] 权限控制生效（无权访问的功能不可见/不可用）
- [ ] 管理员功能正常

### 2.3 数据验证
- [ ] 数据显示正确（无乱码、格式正确）
- [ ] 数据计算正确（公式、汇总、统计）
- [ ] 数据保存正确（刷新后数据不丢失）
- [ ] 数据导入导出正常（如有此功能）

---

## 三、界面验收

### 3.1 界面完整性
- [ ] 所有页面都可以正常访问
- [ ] 页面布局符合设计稿
- [ ] 页面元素完整（无缺失按钮、图片等）

### 3.2 界面美观性
- [ ] 颜色、字体符合设计规范
- [ ] 页面风格统一
- [ ] 无明显的样式问题（错位、重叠等）

### 3.3 交互体验
- [ ] 按钮点击有响应
- [ ] 表单提交有反馈（成功/失败提示）
- [ ] 操作流畅，无明显的卡顿
- [ ] 错误提示友好（用户能理解）

---

## 四、兼容性验收

### 4.1 浏览器兼容
- [ ] Chrome 正常
- [ ] Firefox 正常
- [ ] Safari 正常（如需要）
- [ ] Edge 正常（如需要）

### 4.2 设备兼容
- [ ] PC端正常
- [ ] 平板正常（如需要）
- [ ] 手机正常（如需要）

### 4.3 分辨率适配
- [ ] 1920x1080 正常
- [ ] 1366x768 正常
- [ ] 移动端分辨率正常（如需要）

---

## 五、性能验收

### 5.1 响应速度
- [ ] 页面加载时间 < 3秒
- [ ] 列表查询时间 < 2秒
- [ ] 表单提交时间 < 2秒

### 5.2 并发能力
- [ ] 多用户同时操作无冲突
- [ ] 数据一致性正确

---

## 六、安全验收

### 6.1 认证授权
- [ ] 未登录无法访问系统
- [ ] 密码错误无法登录
- [ ] 登录后权限正确

### 6.2 数据安全
- [ ] 敏感数据已加密或不显示
- [ ] SQL注入防护正常
- [ ] XSS防护正常

---

- [ ] API文档已提供（如需要）
- [ ] 部署文档已提供

## 验收建议

1. **提前准备**：验收前一天确认环境和数据已准备好
2. **按清单执行**：逐项验证，不要凭记忆
3. **记录问题**：发现的问题当场记录，不遗漏
4. **分类处理**：区分严重问题和轻微问题
5. **书面确认**：验收结果需要双方签字确认
