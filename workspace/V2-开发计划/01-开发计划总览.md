# V2开发计划 - 移动端H5

## 一、项目概述

基于V2设计文档，开发移动端H5页面。

### 技术栈
- **框架**: Vue 3 + Vite
- **UI组件**: Vant 4
- **样式**: Tailwind CSS
- **状态管理**: Pinia
- **请求**: Axios
