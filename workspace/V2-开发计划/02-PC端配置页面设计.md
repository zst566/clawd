# PC端配置页面详细设计

## 一、管理后台架构

### 1.1 技术选型
```
前端：Vue 3 + Element Plus + Tailwind CSS
状态管理：Pinia
路由：Vue Router
图表：ECharts（可选）
```

### 1.2 页面结构
```
/admin
  /layout
    - Header.vue（顶部导航）
    - Sidebar.vue（侧边栏）
    - Breadcrumb.vue（面包屑）
  /pages
    /dashboard（首页仪表盘）
    /banners（Banner管理）
    /attractions
      - index.vue（景点列表）
      - detail.vue（景点详情配置）
      - categories.vue（分类管理）
    /homestays
      - index.vue（民宿列表）
      - detail.vue（民宿详情配置）
      - rooms.vue（房间管理）
      - reviews.vue（评价管理）
    /carpool
      - locations.vue（上车点配置）
      - destinations.vue（目的地配置）
      - schedules.vue（班次配置）
      - orders.vue（订单管理）
      - vehicles.vue（车辆管理）
      - drivers.vue（司机管理）
    /articles（资讯管理）
    /users（用户管理）
    /settings
      - cities.vue（城市配置）
      - dicts.vue（数据字典）
```

## 四、API对接

### 4.1 请求封装
```typescript
// request.ts
import axios from 'axios'

const request = axios.create({
  baseURL: '/api/v1',
  timeout: 30000
})

// 请求拦截器
request.interceptors.request.use(config => {
  const token = localStorage.getItem('token')
  if (token) {
    config.headers.Authorization = `Bearer ${token}`
  }
  return config
})

// 响应拦截器
request.interceptors.response.use(
  response => response.data,
  error => {
    // 错误处理
    return Promise.reject(error)
  }
)

export default request
```
