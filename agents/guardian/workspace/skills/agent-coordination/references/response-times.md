# Agent Response Times

## Expected Response Windows

### @zhou_codecraft_bot (CodeCraft)
- **Acknowledgment**: 2-5 minutes
- **Progress update**: 5-10 minutes
- **Task completion**: 
  - Simple: 15-30 minutes
  - Medium: 1-2 hours
  - Complex: 3-6 hours

**Notes**: May need multiple prompts; sometimes replies stay in internal session.

### @zhou_data_bot (Data Assistant)
- **Acknowledgment**: 2-5 minutes
- **Analysis results**: 10-30 minutes
- **Report generation**: 30-60 minutes

**Notes**: Large datasets need more time.

### @guardian (Security)
- **Acknowledgment**: 5-10 minutes
- **Security review**: 30-60 minutes
- **Re-review**: 15-30 minutes

### @inspector (Quality)
- **Acknowledgment**: 5-10 minutes
- **Code review**: 30-60 minutes
- **Re-review**: 15-30 minutes

## Progress Check Timeline

| Time | Action |
|------|--------|
| 5 minutes | Standard progress check |
| 10 minutes | Reminder if no acknowledgment |
| 20 minutes | Mark potential blocker |
| 40 minutes | Escalate or reassign |

## Emergency Escalation

- **30 minutes**: Check if session is active
- **1 hour**: Consider reassignment
- **2 hours**: Escalate to human
