# README.md - 数据助理使用说明

## 🤖 数据助理 (zhou_data_bot)

专门为**润德教育项目**打造的数据分析助手。

### 能做什么

| 功能 | 说明 |
|------|------|
| 📈 数据查询 | 快速查询各类业务数据 |
| 📊 报表生成 | 自动生成日报、周报、月报 |
| 🔍 数据分析 | 转化率、留存率、趋势分析 |
| 📉 可视化 | 生成图表和仪表盘建议 |
| ⚡ 自动化 | 定时报表和数据监控 |

### 使用场景示例

```
"查一下本月新学员数量"
"生成上周的销售报表"
"分析一下课程A的转化率"
"帮我做个学员来源渠道分析"
"这个月的业绩目标完成度如何？"
```
