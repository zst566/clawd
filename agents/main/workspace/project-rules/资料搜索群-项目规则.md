## 核心功能

### 1. 资料搜索
- 根据需求搜索相关资料
- 整理并汇总搜索结果
- 对比分析多个信息源

### 2. 网站验证
- 验证特定网站的可访问性
- 检查网站内容质量
- 评估信息准确性

### 3. 定期资料获取
- 定时从目标网站抓取资料
- 监控内容更新
- 整理并归档

### 4. 技能更新
- 根据获取的资料更新技能库
- 整理有价值的信息

### 5. 安全验证
- 检查目标网站的安全性
- 识别潜在风险
- 提供安全建议

## 定期检查任务

### 检查频率
- **每天一次**

### 检查内容
1. 访问 [awesome-openclaw-skills](https://github.com/VoltAgent/awesome-openclaw-skills) 查看更新
2. 重点关注 **Browser & Automation** (335 skills) 分类
3. 筛选与以下相关的技能：
   - 浏览器自动化测试
   - 回归测试 (regression testing)
   - E2E 测试
   - Playwright / Puppeteer 相关

### 深度分析要求（重要！）
对于发现的每个测试相关技能，必须进行以下分析：

1. **读取技能文档**
   - 获取技能的完整 SKILL.md
   - 理解功能描述和使用方式

2. **外部验证**
   - 通过搜索引擎搜索技能名称 + "review" / "feedback" / "experience"
   - 查找 GitHub issues、Discussions 中的用户反馈
   - 搜索 Reddit、Twitter 等社交媒体的讨论

3. **效果对比**
   - 对比技能描述的功能 vs 实际用户反馈
   - 识别夸大或不足之处
   - 给出客观评分 (1-5⭐)

4. **输出格式**
   ```
   ## 技能分析报告
   - 技能名称: xxx
   - 官方描述: xxx
   - 用户反馈: xxx
   - 实际效果: xxx
   - 推荐指数: ⭐⭐⭐⭐⭐
   ```
