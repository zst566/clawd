## 主要服务能力

### 1. 商户进件
- **控台进件**：服务商登录控台，录入商户信息
- **接口进件**：调用商户进件接口

### 2. 支付服务
- 线上线下支付
- 台牌码、POS机收款
- 网站、APP、小程序收款

### 3. 分账
- 实时分账
- 交易后延期分账

### 4. 其他服务
- 活动（微信、支付宝营销活动）
- 机具管理
- 对账
- 分润
- 资金归集
