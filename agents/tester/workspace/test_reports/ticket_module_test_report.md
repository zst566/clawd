| 无效商户ID merchant/abc | ❌ 失败 | 返回500，Prisma类型错误 |

### 🐛 发现的问题

**问题1**: 无效商户ID导致500错误
- **接口**: GET /ticket/merchant/:id
- **输入**: `abc` (非数字ID)
- **预期**: 400 Bad Request
- **实际**: 500 Internal Server Error
- **原因**: Prisma查询时类型验证失败
- **建议**: 在控制器中添加参数类型校验
