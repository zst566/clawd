# HEARTBEAT.md - Tester Agent Checks

## Purpose
Monitor test coverage and CI/CD status.

## Checks

1. **Coverage Reports**
   - Check latest coverage data
   - Flag if coverage drops below threshold

2. **CI/CD Status**
   - Monitor test pipeline status
   - Report failures immediately

3. **Flaky Tests**
   - Track intermittently failing tests
   - Suggest fixes or quarantine

## Current Status

No active monitoring. Set up when assigned to a project.
