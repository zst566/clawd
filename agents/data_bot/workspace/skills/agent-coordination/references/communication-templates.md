# Communication Templates

## Task Assignment

```
@[agent_name] Start [Phase X - Task Name]:

**Scope**:
1. [Item 1]
2. [Item 2]

**Deliverables**:
- [ ] [Deliverable 1]
- [ ] [Deliverable 2]

**Acceptance Criteria**:
- [Criterion 1]
- [Criterion 2]

**Deadline**: [Time]

Confirm receipt and provide estimated completion time!
```

## 5-Minute Progress Check

```
@[agent_name] ⏰ 5-minute progress check:

**Current Task**: [Task name]

Confirm:
1. Current completion percentage?
2. Any blockers?
3. Can you meet the deadline?

Reply immediately!
```

## Phase Completion

```
@[agent_name] Confirm [Phase X] completion:

Please verify:
- [ ] [Deliverable 1]
- [ ] [Deliverable 2]

If all complete, I'll immediately start [Phase X+1] with @[next_agent]!
```

## Blocker Escalation

### First Reminder (10 min)
```
@[agent_name] ⏰ Second check - No response yet.

Current status? Please reply now or will escalate.
```

### Blocker Marked (20 min)
```
@[agent_name] ⚠️ MARKED BLOCKED

No response for [20] minutes. Task: [Task name]
Escalating to project manager. Please respond!
```
