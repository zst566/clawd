**以"万州分校"为例（2024年全年数据）：**

- **本期余额：** 1,899,164.31

**两者确实不一致，差异原因：**

根据财务逻辑推测：
- **未确认金额**：指该期间内尚未确认收入的预收款金额
- **本期余额**：指截止到该期间结束时的预收款项余额

**计算关系可能是：**
```
未确认金额 = 可确认应收 - 本期确认收入
本期余额 = 累计实收 - 应收可摊金额 - 应收不可摊金额
```

或者更简单的理解：
- **未确认金额** = 某一期的增量（该期新增但未确认的部分）
- **本期余额** = 截止到该期末的累计结余（存量）

这两个是**不同维度的指标**：
- 未确认金额关注"这一期新增了多少还没确认的收入"
- 本期余额关注"到这一期末还剩多少预收款"

这就是为什么两个数字不一样的根本原因。
